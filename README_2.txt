Requerimiento: "En caso de que se intente registrar un código ya existente, se deberá aumentar la cantidad de tomos disponibles."

En lugar de hacer que el administrador tenga que escribir el código nuevamente, añadí los botones para aumentar los tomos disponibles.

Es posible equivocarse de dos maneras distintas:

1. El admin confunde el código de un libro con otro y añade la cantidad de tomos disponibles al libro incorrecto.
2. El admin escribe mal el título y/o autor.

En cualquier caso, el resultado de hacer que el administrador escriba todos los datos para solo añadir la cantidad de libros puede resultar
en errores similares a los que se querían evitar por hacer los controles en papel.

Decidí que el botón para aumentar los tomos disponibles era una solución para lo que se pide.
Separar la lógica de añadir un libro nuevo y uno ya existente permite entonces evitar que el admin use un código ya existente y permite 
añadir tomos disponibles de los libros ya existentes de una manera rápida e intuitiva.
